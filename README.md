A Fusion add-in that adds functionality to import STEP files without uploading
them to the cloud. A button with Burnie as the icon is added in the "Insert"
panel in the Design workspace and the "Manage" panel in the Manufacture
workspace. To use, click the button and select the STEP file(s) you wish to
import - they will be imported as components.

These new components will not be linked, so links do not need to be broken.

To make your life easier with arrangements, disable the "First component
grounded to parent" toggle in your preferences *(General > Design > Assemblies)*.
